# GiftCodeSystem 礼包码系统

- 作者: 淦
- 出处: 无
- 这是一个 TShock 6.1.0 服务器插件，适用于 Terraria 1.4.5，主要用于：管理员发布礼包码，玩家兑换物品。

## 指令

| 语法 | 权限 | 说明 |
| ---- | ---- | ---- |
| /code create <礼包码> <物品列表> [总次数] [每人次数] [过期时间] | code.admin | 创建礼包码 |
| /code delete <礼包码> | code.admin | 删除礼包码 |
| /code list | code.admin | 查看所有礼包码 |
| /code help | code.admin | 查看帮助 |
| /redeem <礼包码> | code.use | 兑换礼包码 |

> 物品列表格式：`物品ID:数量,物品ID:数量`（例如 `1:100,2:50`）  
> 过期时间格式：`YYYY-MM-DD HH:MM`（本地时间），不填表示永久

## 使用方法

- 给管理员分配 `code.admin` 权限，给玩家分配 `code.use` 权限
- 使用 `/code create` 创建礼包码
- 玩家使用 `/redeem` 兑换
- 使用 `/code list` 查看所有礼包码及使用情况
- 使用 `/code delete` 删除不需要的礼包码

## 配置

> 本插件无独立配置文件，所有数据存储在缓存文件中。
